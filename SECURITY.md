## [EN] English 

### Supported Versions
We are currently committed to maintaining and ensuring the active security of the following versions of MediMonitor Pro / ACC.

| Version | Supported          |
| ------- | ------------------ |
| 1.7.0+  | :white_check_mark: |
| < 1.7.0 | :x:                |

### Reporting a Vulnerability

We request that you do **not** report vulnerabilities via public GitHub issues to prevent immediate exploits.

Please report any security issue (e.g. data leaks, auth bypass, SQL injection vulnerabilities) directly to the repository maintainer via contact channels available on their profile ([stordeur-martin.be](https://stordeur-martin.be)). Include as many details as possible (steps to reproduce, affected endpoints or files).

We will try to review every submission within 72 hours, assess the severity, and formulate a patch before publicly disclosing the vulnerability.

## [FR] Français

### Versions Supportées
Nous nous engageons actuellement à maintenir la sécurité des versions suivantes de MediMonitor Pro / ACC.

| Version | Supportée          |
| ------- | ------------------ |
| 1.7.0+  | :white_check_mark: |
| < 1.7.0 | :x:                |

### Signaler une Vulnérabilité

Nous vous demandons de ne **pas** signaler les vulnérabilités de sécurité publiquement via les issues GitHub, afin d'éviter l'exploitation directe des failles.

Signalez tout problème de sécurité (ex. fuite de données, contournement d'authentification SQL injection...) directement au gestionnaire du dépôt via ses contacts disponibles publiquement ([stordeur-martin.be](https://stordeur-martin.be)). Merci d'inclure autant de détails que possible (étapes de reproduction et fichiers impliqués).

Nous ferons de notre mieux pour examiner la proposition sous 72 heures, confirmer sa sévérité et publier un correctif avant de répertorier publiquement la faille.
